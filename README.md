# Autonomous Mobile Robot with Remote Control 

This repository contains the complete implementation of my Bachelor's Thesis project titled:

> Autonomous Mobile Robot with Remote Control

The project focuses on the "design, development, and testing of an autonomous mobile robotic platform** capable of performing multiple intelligent behaviors using onboard sensors and a smartphone-based remote control system.

## Project Overview

The main goal of this project is to design and implement a low-cost autonomous mobile robot platform that integrates:

- Embedded control using Arduino (Makeblock Auriga)
- Sensor-based autonomous navigation
- Bluetooth-based Android smartphone control
- Multiple intelligent operating modes

The system is intended as a practical educational platform for learning and experimenting with mobile robotics, embedded systems, control algorithms, and humanâ€“robot interaction.



---

## Key Features & Functionalities

- Line Following Mode
  - PID-style logic using infrared line sensors
  - Smooth trajectory tracking

- Obstacle Avoidance Mode
  - Ultrasonic distance sensing
  - Dynamic environment scanning and path correction

- Hand Following Mode
  - Real-time human hand tracking using ultrasonic sensing
  - Distance-based motion control

- Remote Control Mode
  - Android smartphone application
  - Bluetooth communication
  - Manual directional control

- Distance Estimation
  - Time-based motion estimation for traveled distance
